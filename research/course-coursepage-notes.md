# Notas de pesquisa para a área de cursos

## Repositório Brumadinho

O repositório `brumadinho_location` descreve uma primeira versão de uma ferramenta para estimar a possível localização de desaparecidos em Brumadinho a partir de latitude, longitude e fluxo de rejeitos. A página pública do repositório indica stack majoritariamente em JavaScript, com HTML e uma pequena camada em Python, além de natureza colaborativa e socialmente orientada.

## PIX via Stripe

A documentação oficial consultada indica que a Stripe oferece suporte a pagamentos com PIX no Brasil, incluindo fluxos de checkout e payment links. O material aponta que o pagamento pode ocorrer por QR Code ou cópia e cola, e que a integração pode ser feita em Checkout, Elements ou API direta. Também há observações específicas sobre IOF e divulgações ao cliente quando a integração é feita pela API.
